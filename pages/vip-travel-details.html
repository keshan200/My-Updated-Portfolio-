Project Overview

The VIP-Travel Management System is a JavaFX-based desktop application designed to revolutionize the car rental experience in Sri Lanka. This system efficiently manages critical business operations, including customer management, user accounts, vehicle inventory, and rental bookings. Additionally, it features integrated reporting capabilities using Jasper Reports for generating detailed invoices and operational reports.

As the developer, I was responsible for designing and implementing the system's architecture, creating an intuitive user interface with JavaFX and Scene Builder, and ensuring seamless integration with the database. The project was carried out as part of my coursework at the Institute of Software Engineering (IJSE).

The system addresses common challenges faced by car rental services, such as efficient resource allocation, accurate billing, and data-driven decision-making, providing a modern solution for enhancing operational efficiency.

Key Features

  • Comprehensive customer, user, and vehicle management
  • Booking system with status tracking and conflict resolution
  • Integration with Jasper Reports for detailed invoice and report generation
  • Secure user authentication and role-based access control
  • Database management using MySQL/PostgreSQL
  • Responsive and user-friendly desktop interface built with JavaFX
  • Dependency management and build automation using Maven
  • Scalable design for future feature enhancements
